# Source Notes: Anomalies and Precision Tests in the Standard Model

These notes are batch-specific extraction context, not canonical graph facts.

## Intended Reading of the Topic

For this batch, `anomalies and precision tests in the Standard Model` should be understood in the standard particle-physics sense:

- anomalies as the breakdown or obstruction of a naive classical symmetry/current story after quantization,
- anomaly cancellation as the consistency condition that keeps the Standard Model gauge structure viable,
- and precision tests as the electroweak-observable layer that checks the theory against high-accuracy measurements.

This batch is intended to support paper, lecture-note, and textbook reading in the Standard Model setting, not a full renormalization-theory or collider-phenomenology reconstruction.

## What the Batch Is Trying to Capture

This batch should identify the smallest defensible prerequisite slice needed to:

- recognize anomaly language and cancellation claims,
- follow the standard electroweak-precision observable vocabulary,
- and keep the anomaly branch separate from the symmetry-breaking and EFT branches.

The batch is not trying to capture:

- all renormalization machinery,
- full electroweak fit methodology,
- collider event modeling,
- or broader BSM anomaly matching.

## Source Guidance

Prioritize standard SM anomaly sections, review notes on current nonconservation, and electroweak-precision passages that explicitly define the observables being compared to experiment.

Prefer a small source set with stable terminology over broad phenomenology references or highly technical fit machinery.

## Anti-Patterns

Avoid:

- collapsing the batch into generic renormalization theory,
- using anomaly language as a synonym for any disagreement with data,
- importing precision-fit details before the observable vocabulary is stable,
- and treating textbook order as the ontology.

## Evidence Guidance

Use evidence that shows:

- how authors define the current or symmetry obstruction,
- where cancellation is required for consistency,
- which observable names are standard in precision tests,
- and how the self-energy / oblique-correction language is introduced.

If a candidate feels more like later EFT or detailed fit machinery, flag it instead of forcing it into the first pass.
